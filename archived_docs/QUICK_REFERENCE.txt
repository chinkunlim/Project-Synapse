├──────────────────────────────────────────────────────────────┤

└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ ✨ 主要功能                                                  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ 📡 系統狀態                                                  │
│    • 即時監控配置狀態                                        │
│    • 測試 API 連接                                           │
│                                                              │
│ 🚀 基本操作                                                  │
│    • 測試 Notion 連接                                        │
│    • 構建儀表板佈局                                          │
│    • 創建數據庫                                              │
│                                                              │
│ ⚡ 快速設置                                                  │
│    • 一鍵初始化（測試→佈局→數據庫）                         │
│                                                              │
│ 🗄️ 數據庫管理                                               │
│    • 列出所有數據庫                                          │
│    • 檢查 Schema 配置                                        │
│    • 顯示環境變數                                            │
│                                                              │
│ ⚠️ 危險區域                                                  │
│    • 清空頁面內容                                            │
│    • 重置所有設置                                            │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 🎯 典型使用場景                                              │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ 場景 1: 首次設置                                             │
│   1. 確認 .env 文件中有 API Key 和 Page ID                  │
│   2. 點擊「一鍵初始化所有資料庫」                            │
│   3. 等待完成，檢查日誌                                      │
│                                                              │
│ 場景 2: 日常檢查                                             │
│   1. 點擊「測試 Notion 連接」                                │
│   2. 查看系統狀態面板                                        │
│                                                              │
│ 場景 3: 重新開始                                             │
│   1. 點擊「重置所有設置」                                    │
│   2. 確認警告提示                                            │
│   3. 等待自動重建完成                                        │
│                                                              │
│ 場景 4: 檢查配置                                             │
│   1. 點擊「檢查 Schema 配置」                                │
│   2. 點擊「顯示環境變數」                                    │
│   3. 點擊「列出所有數據庫」                                  │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 🎨 介面元素說明                                              │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ 顏色標示:                                                    │
│   🟢 綠色 - 成功狀態 / 操作成功                              │
│   🔵 藍色 - 信息狀態 / 一般訊息                              │
│   🟡 黃色 - 警告狀態 / 需要注意                              │
│   🔴 紅色 - 錯誤狀態 / 危險操作                              │
│                                                              │
│ Badge 狀態:                                                  │
│   ✅ 已設定 - 配置項已正確設置                               │
│   ❌ 未設定 - 配置項缺失                                     │
│   ⏳ 待初始化 - 需要執行初始化操作                           │
│                                                              │
│ Loading 動畫:                                                │
│   • 全屏覆蓋，顯示處理中                                     │
│   • 動畫 spinner + 操作說明                                  │
│                                                              │
│ 操作日誌:                                                    │
│   • 顯示所有操作的執行結果                                   │
│   • 帶時間戳記錄                                             │
│   • 可清空日誌                                               │
│   • 自動滾動到最新                                           │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ ⚠️ 重要提醒                                                  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ 1. 環境變數必須設置:                                         │
│    • NOTION_API_KEY                                          │
│    • PARENT_PAGE_ID                                          │
│                                                              │
│ 2. 危險操作會要求二次確認                                    │
│                                                              │
│ 3. 所有操作都會記錄在日誌中                                  │
│                                                              │
│ 4. 敏感信息會自動遮罩顯示                                    │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ 🐛 故障排除                                                  │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ 問題: 連接失敗                                               │
│ 解決: 檢查 NOTION_API_KEY 是否正確                           │
│                                                              │
│ 問題: Parent Page ID 未設置                                  │
│ 解決: 在 .env 文件中添加 PARENT_PAGE_ID                      │
│                                                              │
│ 問題: Schema 文件錯誤                                         │
│ 解決: 檢查 config/notion_schema.json 是否存在且格式正確      │
│                                                              │
│ 問題: 端口被佔用                                             │
│ 解決: 修改 app.py 中的端口號或停止佔用進程                   │
│                                                              │
